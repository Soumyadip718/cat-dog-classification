# Cat and Dog Classification using Deep Learning

This project is a Deep Learning based image classification system that classifies Cats and Dogs using TensorFlow/Keras.

## Features
- Image Classification
- CNN Model
- Training and Testing
- Accuracy Visualization

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Dataset
TensorFlow Cats and Dogs Filtered Dataset

## How to Run

1. Install dependencies
```bash
pip install -r requirements.txt
```

2. Run the notebook
```bash
jupyter notebook
```
